# Description

This repository contains a **graphical interface** project used to display a **personal herbarium**. The GUI is written in Python and is based on QT libraries, offering the following features:

* **Dynamic workspace** where the data is stored.
* **Botany classification** tree view with species adding interface.
* **Species view** with basic informations and books references.
* **Identifications list** with date, place, description and associated photos.

# Usage

## Local run

```bash
git clone https://github.com/Ludovic-Lesur/botany-guide-gui.git
cd botany-guide-gui
python3 -m gui.gui
```

## Executable

The binaries are available for Ubuntu 22.04 and Windows in the [Releases page](https://github.com/Ludovic-Lesur/botany-guide-gui/releases)
